# bee-geo 产品需求文档

## 1. 产品定位

bee-geo 是企业级 GEO 分析与 AI 内容运营平台，帮助内容、市场、销售和品牌团队完成问题洞察、内容生成、审核排期、自动发布和效果归档。

## 2. 用户角色

| 角色 | 核心职责 |
| --- | --- |
| 超级管理员 | 管理组织、用户、角色、授权凭据、系统配置 |
| 内容管理员 | 管理关键词、知识库、素材、AI 人设、内容草稿 |
| 审核员 | 审核内容，决定是否允许排期发布 |
| 发布员 | 管理发布账号、排期任务、失败重试和人工介入 |
| 只读观察员 | 查看报表、GEO 结果和发布回执 |

## 3. 功能模块

### 3.1 总览仪表盘

- 展示 GEO 任务数、内容产出、待审核数量、发布成功率、失败任务和账号异常。
- 提供待处理事项入口：待审核内容、失败发布任务、授权过期账号。

### 3.2 关键词库

- 支持关键词分组、关键词新增、编辑、删除、批量上传。
- 关键词可作为 GEO 分析和 AI 创作输入。

### 3.3 知识库

- 支持文档新增、替换、分组、类型管理。
- 文档类型包含 `txt`、`md`、常见办公文件。
- 知识库作为 AI 创作和人设生成的上下文资产。

### 3.4 素材库

- 支持图片、视频、表格、活动物料管理。
- 支持标签分类、上传入口、编辑删除。
- 素材可在内容创作和发布封面中复用。

### 3.5 GEO 分析

- 输入关键词生成模拟问题。
- 展示任务状态、时间、操作人。
- 结果页展示问题蒸馏词、模拟问题、AI 引用分析、引用来源 URL、媒体名称和内容描述。
- 支持从结果页进入 AI 创作。

### 3.6 AI 创作

- 支持选择品牌、发布平台、文章类型、提示词和 AI 人设。
- 支持生成标题、摘要、正文、关键词。
- 支持保存草稿、提交审核、保存并进入发送设置。

### 3.7 AI 人设

- 支持上传文件或输入文本生成 AI 人设。
- 支持结构化编辑角色定位、表达重心、核心视角、情感温度。
- 人设可被 AI 创作引用。

### 3.8 发布中心

- 支持发布任务创建、审核后排期、执行、失败重试、人工介入、撤回和回执归档。
- 第一版发布渠道覆盖自有站点和免费媒体。
- 发布失败有限自动重试，超过阈值转人工处理。

### 3.9 集成设置

- 支持自有站点授权：名称、地址、授权 Token。
- 支持免费媒体授权：媒体名称、Cookie、授权状态。
- 凭据必须加密存储、脱敏展示、过期提醒。

### 3.10 用户管理与审计

- 支持用户、账号、角色、管理员类型管理。
- 审计覆盖登录、授权、生成、编辑、审核、排期、发布、重试、撤回、删除。

## 4. 内容状态

```text
草稿 -> 待审核 -> 审核通过 -> 已排期 -> 发布中 -> 发布成功
                              └-> 发布失败 -> 人工介入 -> 重试
待审核 -> 已驳回
已排期 -> 已撤回
```

## 5. 验收标准

- 能用模拟数据完整演示从 GEO 分析到自动化发布的主流程。
- 所有高频变化点具备接口、配置或适配器隔离。
- 关键操作均具备权限控制和审计设计。
